# Amazon Sales Analysis

## Project Overview
This project analyzes Amazon product sales data using Python and Pandas. It explores product categories, ratings, discounts, prices, and customer reviews through data cleaning and visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Amazon Sales Dataset

## Analysis Performed
- Checked dataset information
- Checked missing values
- Removed duplicate records
- Cleaned price, rating, and discount columns
- Analyzed product categories
- Visualized rating distribution
- Compared actual price and discounted price
- Identified top-rated and most-reviewed products

## Visualizations
- Top 10 Product Categories
- Average Rating by Category
- Top 10 Highest Rated Products
- Discount Percentage Distribution
- Actual Price vs Discounted Price
- Top 10 Most Reviewed Products

## Conclusion
The analysis provides insights into Amazon products by highlighting popular categories, customer ratings, pricing trends, and discount patterns. It demonstrates the use of Python for data cleaning, analysis, and visualization.
